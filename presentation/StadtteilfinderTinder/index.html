Open Data Day 2016

Frankfurt

  • 16 area districts (Ortsbezirke)
  • 46 city districts (Stadtteile)
  • 118 city boroughs (Stadtbezirke)

… where would you want to live?

Setup

  • randomly pick a Frankfurt address
    • from Hauskoordinaten open data
  • present the user with a photo of the address
    • from the Google Streetview API
  • user "swipes" left ("don't like"), or right ("like")
    • store all swipes on the server
    • present the "district match", and a heatmap of "district love"

Technology used

  • node.js for the server
  • socket.io for server/client communication
  • amazon aws for the actual VM (free version)
  • postgreSQL for database storing
  • R for calculations and mapping
  • plumbeR for RESTful communication between R and node.js
  • Ebbelwoi for the idea
